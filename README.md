커피의 온도 변화


* Formula
```r
T = 커피온도

dT/dt = -k(T-Troom)
T(t) = Troom + (T0 - Troom)e^(-kt)
```

* Forward Euler
```r
T[n+1] = T[n] - k*dt*(T[n] - Troom)
```

* Backward Euler
```r
T[n+1] = (T[n] + k*dt*Troom)/(1 + k*dt)
```

* Trapezoidal
```r
T[n+1] = (T[n]*(1 - k*dt/2) + k*dt*Troom)/(1 + k*dt/2)
```

* Gear's method (BDF)
